# Utendørsliv Landing Page

Professional landing page for the Utendørsliv mobile app.

## Features
- **Modern Design:** Nature-inspired palette and responsive layout.
- **Legal Pages:** Privacy Policy and Support pages included.
- **Auto-Deploy:** Automatically deployed to GitHub Pages via GitHub Actions.

## Development
To view the site locally, just open `index.html` in any browser.

## Deployment
Any push to the `main` branch will trigger a GitHub Action that deploys the static files to the `gh-pages` branch.
